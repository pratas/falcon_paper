<p align="justify">
Auxiliar <b>tools to compute the results of the paper describing FALCON</b>. It requires a <b>Linux</b> OS with <b>cmake</b>, <b>git</b> and <b>gnuplot</b>. 
</p>

Install:
<pre>
git clone https://github.com/pratas/falcon_paper.git
cd falcon_paper/
</pre>

Running:
<pre>
. runSubs.sh
. runPerm.sh
</pre>
