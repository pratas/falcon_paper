Auxiliar tools to compute the results of FALCON paper. It requires a Linux OS with cmake

Install:
<pre>
git clone https://github.com/pratas/falcon_paper.git
cd falcon_paper/
</pre>

Running:
<pre>
. runSubs.sh
. runPerm.sh
</pre>
